##########RepoInfoApp###################
Execution steps:
1. Download or clone this repo
2. go to repoinfo and execute the following:
	mvn clean install
	./mvnw spring-boot:run
3. Check if http://localhost:8080/repos/kubernetes/report is displaying
4. go to repo-app and execute the following:
	npm start
5. Check if localhost:9000 is working
